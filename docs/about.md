# Acerca de este sitio

Este sitio está diseñado como un **punto de partida** para aprender sobre **forensia digital**.

## ¿Por qué este proyecto?

El objetivo es proporcionar una **referencia rápida** que pueda desplegarse fácilmente como una **GitHub Page** usando MkDocs.

## Cómo usarlo

1. Edita los archivos markdown en `docs/` para ajustar o ampliar el contenido.
2. Usa `mkdocs serve` para ver los cambios localmente.
3. Genera el sitio con `mkdocs build` y despliega en GitHub Pages.
